# 🌸 Склад Цветочного Магазина

Полноценное веб-приложение для управления складом цветочного магазина.
**Node.js + Express + SQLite3 + Vanilla JS**

---

## 📁 Структура проекта

```
flower-shop/
├── src/
│   ├── server.js               ← Точка входа (Express)
│   ├── db/
│   │   └── database.js         ← SQLite3 схема + сид данных
│   ├── middleware/
│   │   └── auth.js             ← Проверка сессии / роли
│   └── routes/
│       ├── auth.js             ← Вход / выход / смена пароля
│       ├── admin.js            ← Управление пользователями
│       └── inventory.js        ← Товары / движения / статистика
├── public/
│   ├── login.html              ← Страница входа
│   ├── index.html              ← Главное SPA
│   ├── css/
│   │   └── app.css
│   ├── js/
│   │   └── app.js
│   └── uploads/                ← Фото товаров (создаётся авто)
├── data/
│   └── warehouse.db            ← База данных (создаётся авто)
└── package.json
```

---

## 🚀 Запуск

### 1. Установить зависимости
```bash
cd flower-shop
npm install
```

### 2. Запустить сервер
```bash
npm start
# или для разработки с авто-перезапуском:
npm run dev
```

### 3. Открыть браузер
```
http://localhost:3000
```

## ✨ Возможности

### 📦 Склад
- Добавление, редактирование, удаление товаров
- Фото товара (загрузка / drag & drop)
- Быстрый приход/расход (+/− прямо в таблице)
- Полноценный модал прихода/расхода с примечанием
- Вид: таблица или плитка
- Фильтр по категории, статусу, поиск
- Экспорт в CSV

### 📊 Дашборд
- KPI: позиции, штуки, стоимость, предупреждения
- График движений за 7 дней
- Топ товаров по остатку
- Лента последних действий
- Разбивка по категориям
- Сезонный совет (по месяцу)

### 📈 Отчёты
- Периоды: 7, 14, 30 дней
- График по дням
- История всех движений
- Кто и когда изменил товар

### 👥 Пользователи (только для Администратора)
- Создание новых пользователей (admin / user)
- Сброс пароля любого пользователя
- Деактивация / активация аккаунта
- Удаление пользователя

### 🔑 Безопасность
- Пароли хешируются через bcrypt
- Сессии хранятся в SQLite
- Роли: admin / user
- Защита всех API роутов middleware
